# Directory Explorer Tests

This repository contains unit tests for the **Directory Explorer** application. The tests verify the functionality and correctness of the application's directory exploration features.

## Repository Structure

```
├── DirectoryExplorer.Tests.csproj
├── DirectoryExplorerTests.cs
└── README.md
```

## Files

- **DirectoryExplorer.Tests.csproj** – C# test project configuration.
- **DirectoryExplorerTests.cs** – Contains unit test cases for the Directory Explorer application.

## Features

- Unit testing for directory exploration functionality
- Validation of expected behavior
- Automated test execution using a .NET testing framework

## Technologies Used

- C#
- .NET
- Unit Testing Framework (MSTest/xUnit/NUnit)

## Objective

- Verify the correctness of the Directory Explorer application.
- Ensure reliable and maintainable code through automated testing.
