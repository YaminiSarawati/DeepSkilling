# Player Manager Library

This repository contains the core implementation of the **Player Manager Library**, a reusable C# class library for managing player information.

## Repository Structure

```text
├── PlayerManager.cs
├── PlayersManagerLib.csproj
└── README.md
```

## Files

- **PlayerManager.cs** – Implements the player management logic.
- **PlayersManagerLib.csproj** – C# class library project configuration.

## Features

- Player information management
- Add, update, remove, and retrieve player records
- Reusable .NET class library
- Clean and modular implementation

## Technologies Used

- C#
- .NET Class Library

## Objective

- Build a reusable library for player management.
- Demonstrate object-oriented programming concepts.
- Enable integration into other .NET applications.
