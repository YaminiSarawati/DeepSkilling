# NUnit Testing Projects

This repository contains C# class libraries and their corresponding NUnit test projects. The projects demonstrate library development and automated unit testing using the .NET framework.

## Repository Structure

```text
├── DirectoryExplorer.Tests/
├── MagicFilesLib/
├── PlayerManager.Tests/
├── PlayerManagerLib/
├── CalcLibrary.Tests/
├── CalcLibrary/
├── CustomerComm.Tests/
├── CustomerCommLib/
└── README.md
```

## Projects

### Libraries
- **MagicFilesLib** – Directory exploration and file management library.
- **PlayerManagerLib** – Library for managing player information.
- **CalcLibrary** – Reusable library for performing mathematical operations.
- **CustomerCommLib** – Library for handling customer email communication.

### Test Projects
- **DirectoryExplorer.Tests** – Unit tests for the Directory Explorer library.
- **PlayerManager.Tests** – Unit tests for the Player Manager library.
- **CalcLibrary.Tests** – Unit tests for the Calculator Library.
- **CustomerComm.Tests** – Unit tests for the Customer Communication Library.

## Features

- Reusable C# class libraries
- Automated unit testing using NUnit
- Modular and maintainable project structure
- Demonstrates best practices for testing .NET applications

## Technologies Used

- C#
- .NET
- NUnit Testing Framework

## Learning Objectives

- Build reusable class libraries in C#.
- Write and execute unit tests using NUnit.
- Improve code quality through automated testing.
- Understand test-driven development (TDD) concepts.
