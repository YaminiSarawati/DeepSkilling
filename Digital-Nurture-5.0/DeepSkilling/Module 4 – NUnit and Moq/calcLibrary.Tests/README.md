# Calculator Library Tests

This repository contains unit tests for the **Calculator Library**. The tests verify the correctness and reliability of the calculator's arithmetic operations.

## Repository Structure

```text
├── CalcLibrary.Tests.csproj
├── SimpleCalculatorTests.cs
└── README.md
```

## Files

- **CalcLibrary.Tests.csproj** – C# test project configuration.
- **SimpleCalculatorTests.cs** – Contains unit test cases for the Calculator Library.

## Features

- Unit testing for calculator operations
- Validation of arithmetic functions
- Automated test execution using a .NET testing framework

## Technologies Used

- C#
- .NET
- Unit Testing Framework (MSTest/xUnit/NUnit)

## Objective

- Verify the correctness of calculator operations.
- Ensure reliable and maintainable code through automated testing.
- Demonstrate best practices for unit testing in .NET.
