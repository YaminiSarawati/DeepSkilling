# Calculator Library

This repository contains the implementation of a simple **Calculator Library** developed in C#. It provides reusable mathematical operations that can be integrated into .NET applications.

## Repository Structure

```text
├── CalcLibrary.csproj
├── MathLibrary.cs
└── README.md
```

## Files

- **CalcLibrary.csproj** – C# class library project configuration.
- **MathLibrary.cs** – Contains the implementation of mathematical and arithmetic operations.

## Features

- Basic arithmetic operations
- Reusable .NET class library
- Clean and modular code structure
- Easy integration with other C# applications

## Technologies Used

- C#
- .NET Class Library

## Objective

- Develop a reusable calculator library.
- Demonstrate object-oriented programming concepts in C#.
- Provide mathematical functionality for .NET applications.
