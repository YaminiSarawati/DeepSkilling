# Customer Communication Library

This repository contains the implementation of the **Customer Communication Library**, a reusable C# class library for handling customer communication functionalities.

## Repository Structure

```text
├── CustomerCommLib.csproj
├── MailSender.cs
└── README.md
```

## Files

- **CustomerCommLib.csproj** – C# class library project configuration.
- **MailSender.cs** – Implements the email sending and customer communication functionality.

## Features

- Email sending functionality
- Reusable .NET class library
- Clean and modular implementation
- Easy integration with other C# applications

## Technologies Used

- C#
- .NET Class Library

## Objective

- Develop a reusable library for customer communication.
- Demonstrate object-oriented programming concepts in C#.
- Provide email communication functionality for .NET applications.
