# Product Catalog Models

This repository contains C# model classes for a simple **Product Catalog** application. It includes entity definitions for products and categories, along with a sample output image.

## Repository Structure

```text
├── Category.cs
├── Product.cs
├── lab1.png
└── README.md
```

## Files

- **Category.cs** – Defines the category entity and its properties.
- **Product.cs** – Defines the product entity and its associated details.
- **lab1.png** – Sample output or project demonstration image.

## Features

- Product and category model classes
- Object-oriented design using C#
- Reusable entity definitions
- Sample project output included

## Technologies Used

- C#
- .NET

## Learning Objectives

- Understand entity modeling in C#.
- Practice object-oriented programming principles.
- Build reusable models for business applications.
