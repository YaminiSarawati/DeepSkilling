# Entity Framework DbContext

This repository demonstrates the implementation of an **Entity Framework DbContext** for managing database interactions in a C# application.

## Repository Structure

```text
├── AppDbContext.cs
├── lab2.png
└── README.md
```

## Files

- **AppDbContext.cs** – Defines the application's database context and manages entity-to-database mapping.
- **lab2.png** – Sample output or project demonstration image.

## Features

- Entity Framework DbContext implementation
- Database connection and context management
- Foundation for CRUD operations
- Sample project output included

## Technologies Used

- C#
- .NET
- Entity Framework Core

## Learning Objectives

- Understand the role of `DbContext` in Entity Framework Core.
- Learn how to configure and manage database contexts.
- Build a foundation for database-driven .NET applications.
