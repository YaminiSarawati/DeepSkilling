# Entity Framework Core Migrations

This repository contains the initial **Entity Framework Core migration** files used to create and manage the application's database schema.

## Repository Structure

```text
├── 20260711162843_InitialCreate.cs
├── 20260711162843_InitialCreate.Designer.cs
├── AppDbContextModelSnapshot.cs
├── lab31.png
├── lab32.png
└── README.md
```

## Files

- **20260711162843_InitialCreate.cs** – Initial migration containing the database schema changes.
- **20260711162843_InitialCreate.Designer.cs** – Designer file generated for the initial migration.
- **AppDbContextModelSnapshot.cs** – Snapshot of the current Entity Framework Core model.
- **lab31.png** – Screenshot demonstrating the migration process or output.
- **lab32.png** – Additional screenshot of the migration results.

## Features

- Initial Entity Framework Core migration
- Database schema versioning
- Model snapshot for change tracking
- Sample screenshots of migration results

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Database

## Learning Objectives

- Understand Entity Framework Core migrations.
- Learn how to create and apply database migrations.
- Track schema changes using model snapshots.
- Manage database versioning in .NET applications.
