# Entity Framework Core Console Application

This repository contains a C# console application that demonstrates the use of **Entity Framework Core** for interacting with a database.

## Repository Structure

```text
├── Program.cs
├── lab4.png
├── lab41.png
├── lab42.png
└── README.md
```

## Files

- **Program.cs** – Main application that demonstrates Entity Framework Core operations.
- **lab4.png** – Screenshot of the application output.
- **lab41.png** – Screenshot showing database operations or results.
- **lab42.png** – Additional screenshot demonstrating the application execution.

## Features

- Entity Framework Core integration
- Console-based application
- Database interaction examples
- Sample output screenshots

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Database

## Learning Objectives

- Understand how to use Entity Framework Core in a console application.
- Perform basic database operations using EF Core.
- Learn the workflow of connecting, querying, and managing data.
