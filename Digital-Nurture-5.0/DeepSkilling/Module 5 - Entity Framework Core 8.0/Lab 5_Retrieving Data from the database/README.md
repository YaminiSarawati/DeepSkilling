# Entity Framework Core CRUD Operations

This repository contains a C# console application demonstrating **CRUD (Create, Read, Update, Delete)** operations using **Entity Framework Core**.

## Repository Structure

```text
├── Program.cs
├── lab5.png
└── README.md
```

## Files

- **Program.cs** – Main application implementing CRUD operations with Entity Framework Core.
- **lab5.png** – Screenshot of the application output.

## Features

- Create records
- Read data from the database
- Update existing records
- Delete records
- Console-based application

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Database

## Learning Objectives

- Understand CRUD operations using Entity Framework Core.
- Learn how to interact with a database from a C# console application.
- Practice implementing database operations with EF Core.
