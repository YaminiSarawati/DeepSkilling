# Entity Framework Core Labs

This repository contains a series of hands-on labs demonstrating the fundamentals of **Entity Framework Core (EF Core)** using a Retail Inventory System. The labs cover entity modeling, database context configuration, migrations, data insertion, and data retrieval.

## Repository Structure

```text
├── Lab 1_Understanding ORM with a Retail Inventory System/
├── Lab 2_Setting Up the Database Context for a Retail Store/
├── Lab 3_Using EF Core CLI to Create and Apply Migrations/
├── Lab 4_Inserting Initial Data into the Database/
├── Lab 5_Retrieving Data from the Database/
└── README.md
```

## Labs Overview

### Lab 1 – Understanding ORM with a Retail Inventory System
- Create Product and Category entities
- Understand Object-Relational Mapping (ORM)
- Design a simple retail inventory model

### Lab 2 – Setting Up the Database Context for a Retail Store
- Configure `AppDbContext`
- Manage database connections
- Map entities using Entity Framework Core

### Lab 3 – Using EF Core CLI to Create and Apply Migrations
- Generate initial migrations
- Apply database schema changes
- Understand model snapshots and versioning

### Lab 4 – Inserting Initial Data into the Database
- Insert sample records into the database
- Use Entity Framework Core for data persistence
- Verify inserted data

### Lab 5 – Retrieving Data from the Database
- Query data using Entity Framework Core
- Retrieve and display records
- Practice basic data retrieval operations

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Database

## Learning Objectives

- Understand Object-Relational Mapping (ORM).
- Configure and use Entity Framework Core.
- Create and apply database migrations.
- Perform data insertion and retrieval operations.
- Build database-driven .NET applications using EF Core.
