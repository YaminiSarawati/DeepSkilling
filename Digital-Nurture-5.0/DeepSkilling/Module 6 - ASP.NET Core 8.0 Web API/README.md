# ASP.NET Core Web API Hands-On

This repository contains a hands-on project demonstrating the development of an **ASP.NET Core Web API**. It covers RESTful API creation, controller implementation, authentication, custom filters, configuration management, and project organization.

## Repository Structure

```text
├── Controllers/
├── Filters/
├── Models/
├── Properties/
├── .gitkeep
├── Program.cs
├── WeatherForecast.cs
├── WebApi_Handson.csproj
├── WebApi_Handson.http
├── appsettings.json
├── appsettings.Development.json
└── README.md
```

## Project Modules

### Controllers
- Authentication API
- Employee API
- Values API
- Weather Forecast API

### Filters
- Custom Authentication Filter
- Custom Exception Filter

### Models
- Employee entity model

### Properties
- Launch settings for application startup

### Configuration Files
- **appsettings.json** – General application configuration.
- **appsettings.Development.json** – Development environment settings.
- **WebApi_Handson.http** – HTTP requests for testing API endpoints.
- **.gitkeep** – Keeps empty directories under version control.

## Features

- RESTful Web API development
- Authentication and authorization
- Custom authentication and exception filters
- Employee management endpoints
- Weather forecast sample service
- Environment-based configuration
- API endpoint testing support

## Technologies Used

- C#
- ASP.NET Core Web API
- .NET
- REST API
- JSON Configuration

## Learning Objectives

- Develop RESTful APIs using ASP.NET Core.
- Understand controller-based application architecture.
- Implement authentication and custom filters.
- Configure applications using `appsettings.json`.
- Test APIs using HTTP request files.
