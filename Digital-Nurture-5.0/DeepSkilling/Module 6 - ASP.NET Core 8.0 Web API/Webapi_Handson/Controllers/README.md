# ASP.NET Core Web API Controllers

This repository contains the controller implementations for an **ASP.NET Core Web API**. The controllers demonstrate authentication, employee management, API endpoints, and weather forecast services.

## Repository Structure

```text
├── AuthController.cs
├── EmployeeController.cs
├── ValuesController.cs
├── WeatherForecastController.cs
└── README.md
```

## Files

- **AuthController.cs** – Handles user authentication and authorization.
- **EmployeeController.cs** – Manages employee-related API operations.
- **ValuesController.cs** – Provides sample API endpoints for testing and demonstration.
- **WeatherForecastController.cs** – Returns weather forecast data through REST API endpoints.

## Features

- User authentication
- Employee management APIs
- RESTful API endpoints
- Weather forecast service
- ASP.NET Core MVC controller implementation

## Technologies Used

- C#
- ASP.NET Core Web API
- .NET
- REST API

## Learning Objectives

- Understand ASP.NET Core Web API controllers.
- Build RESTful API endpoints.
- Implement authentication and authorization.
- Develop controller-based services for web applications.
