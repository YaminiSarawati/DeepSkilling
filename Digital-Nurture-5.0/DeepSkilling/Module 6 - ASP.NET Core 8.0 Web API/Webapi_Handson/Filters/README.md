# ASP.NET Core Custom Filters

This repository contains custom filter implementations for an **ASP.NET Core Web API**. The filters demonstrate how to implement custom authentication and centralized exception handling.

## Repository Structure

```text
├── CustomAuthFilter.cs
├── CustomExceptionFilter.cs
└── README.md
```

## Files

- **CustomAuthFilter.cs** – Implements a custom authentication filter to validate incoming requests.
- **CustomExceptionFilter.cs** – Implements a custom exception filter for centralized error handling and response generation.

## Features

- Custom authentication filter
- Global exception handling
- Improved API security
- Centralized error management
- Reusable filter components

## Technologies Used

- C#
- ASP.NET Core
- .NET
- Web API Filters

## Learning Objectives

- Understand the ASP.NET Core filter pipeline.
- Implement custom authentication using action filters.
- Handle exceptions globally with exception filters.
- Improve API security and maintainability.
