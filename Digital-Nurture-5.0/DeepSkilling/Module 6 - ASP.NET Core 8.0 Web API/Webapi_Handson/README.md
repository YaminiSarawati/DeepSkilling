# ASP.NET Core Web API Hands-On

This repository contains a hands-on implementation of an **ASP.NET Core Web API**. The project demonstrates RESTful API development, authentication, custom filters, configuration management, and sample endpoints.

## Repository Structure

```text
├── Controllers/
├── Filters/
├── Models/
├── Properties/
├── Program.cs
├── WeatherForecast.cs
├── WebApi_Handson.csproj
├── WebApi_Handson.http
├── appsettings.json
├── appsettings.Development.json
└── README.md
```

## Project Components

### Controllers
- Authentication endpoints
- Employee management APIs
- Sample Values API
- Weather Forecast API

### Filters
- Custom Authentication Filter
- Custom Exception Filter

### Models
- Employee model

### Properties
- Launch settings for development

### Configuration
- **appsettings.json** – Application configuration.
- **appsettings.Development.json** – Development-specific configuration.
- **WebApi_Handson.http** – HTTP request file for testing API endpoints.

## Features

- RESTful Web API development
- Authentication and authorization
- Custom filters for authentication and exception handling
- Employee management endpoints
- Weather forecast sample API
- Environment-based configuration
- HTTP endpoint testing support

## Technologies Used

- C#
- ASP.NET Core Web API
- .NET
- REST API
- JSON Configuration

## Learning Objectives

- Build RESTful APIs using ASP.NET Core.
- Implement authentication and authorization.
- Create and use custom filters.
- Manage application configuration using appsettings files.
- Test API endpoints using HTTP request files.
