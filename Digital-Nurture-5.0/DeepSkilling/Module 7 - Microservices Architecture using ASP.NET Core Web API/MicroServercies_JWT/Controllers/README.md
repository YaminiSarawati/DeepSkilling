# ASP.NET Core Secure Web API Controllers

This repository contains controller implementations for an **ASP.NET Core Secure Web API**. The controllers demonstrate authentication, role-based authorization, protected endpoints, and sample weather forecast services.

## Repository Structure

```text
├── AdminController.cs
├── AuthController.cs
├── SecureController.cs
├── WeatherForecastController.cs
└── README.md
```

## Files

- **AdminController.cs** – Provides administrator-only API endpoints with role-based authorization.
- **AuthController.cs** – Handles user authentication and token generation.
- **SecureController.cs** – Contains protected API endpoints that require authenticated access.
- **WeatherForecastController.cs** – Provides sample weather forecast API endpoints.

## Features

- User authentication
- Role-based authorization
- Protected API endpoints
- Admin-specific operations
- RESTful API controller implementation
- Weather forecast sample service

## Technologies Used

- C#
- ASP.NET Core Web API
- .NET
- REST API
- JWT Authentication (if implemented)

## Learning Objectives

- Understand authentication and authorization in ASP.NET Core.
- Implement secure RESTful API endpoints.
- Apply role-based access control.
- Build controller-based web services.
