# User Model

This repository contains the **User** model class used in an **ASP.NET Core Web API** application. The model represents user information for authentication, authorization, and application-level user management.

## Repository Structure

```text

```

- User data model
- Supports authentication and authorization
- Reusable entity for ASP.NET Core applications
- Object-oriented class design

## Technologies Used

- C#
- .NET
- ASP.NET Core
