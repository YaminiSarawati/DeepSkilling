# ASP.NET Core Launch Settings

This repository contains the **launch settings** configuration for an ASP.NET Core application. The configuration defines how the application is run during development, including launch profiles, environment variables, application URLs, and IIS Express settings.

## Repository Structure

```text
├── launchSettings.json
└── README.md
```

## Files

- **launchSettings.json** – Configures launch profiles, application URLs, environment variables, and development settings for the ASP.NET Core application.

## Features

- Multiple launch profiles
- Development environment configuration
- IIS Express support
- Application URL configuration
- Environment variable management

## Technologies Used

- ASP.NET Core
- .NET
- JSON

## Learning Objectives

- Understand the purpose of `launchSettings.json`.
- Configure launch profiles for ASP.NET Core applications.
- Manage development environment settings.
- Customize application startup during development.
